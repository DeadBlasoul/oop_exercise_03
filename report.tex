\subsection*{Условие}

Задание №1: написать программу, которая реализует работу с фигурами: 
\begin{enumerate}
\item Вставка
\item Удаление
\item Печать фигуры
\item Печать всех фигур
\end{enumerate}

\subsection*{Описание программы}

Код программы состоит из 5-ти файлов:
\begin{enumerate}
\item app/main.cpp: исходный код с точкой входа
\item src/figure.hpp: объявление и реализация generic структур
\item src/figure.cpp: реализация функций
\item src/point.hpp: объявление и реализация generic структур
\item src/point.cpp: реализация функций
\end{enumerate}
